Clase vInt


Ámbito privado:

vRef data: referencia en vHeap al dato que contiene la clase



Ámbito publico:

vInt();
Proceso realizado:
Construye la clase con valor por defecto en 0.

vInt(int);
Parámetros:
int: variable a guardar en vHeap.
Proceso realizado:
Hace un vMalloc para crear la referencia data y un vPlacement para llenarla con el parámetro.

~vInt();
Proceso realizado:
Destruye la referencia al dato en vHeap.

int operator =(int);
Parámetros:
int: variable a la que se está igualando.
Proceso realizado:
Asigna el parametro al valor almacenado en la vRef.

int operator =(vInt);
Parámetros:
vInt: variable a la que se está igualando.
Proceso realizado:
Asigna el parametro al valor almacenado en la vRef. Retorna 0 si todo sale bien.

int operator !();
Proceso realizado:
Des referencia el dato contenido el la vRef y lo retorna.

int operator +=(int);
Parámetros:
int: variable que se esta sumando
Proceso realizado:
Suma el parámetro al dato apuntado por la vRef. Retorna 0 si todo sale bien.

vRef operator &();
Proceso realizado:
Retorna la vRef corresondiente al dato almacenado.

int operator --();
Proceso realizado:
Resta 1 a la variable apuntada por la vRef. Retorna 0 si todo sale bien.

int operator ++();
Proceso realizado:
Suma 1 a la variable apuntada por la vRef. Retorna 0 si todo sale bien.

bool operator <(vInt);
Parámetros:
vInt: variable con la cual se va a comparar
Proceso realizado:
Compara los datos de las vRef. Retorna true si el parámetro es mayor.

bool operator <=(vInt);
Parámetros:
vInt: variable con la cual se va a comparar
Proceso realizado:
Compara los datos de las vRef. Retorna true si el parámetro es mayor o igual.

bool operator >(vInt);
Parámetros:
vInt: variable con la cual se va a comparar

bool operator >=(vInt);
Parámetros:
vInt: variable con la cual se va a comparar
Proceso realizado:
Compara los datos de las vRef. Retorna true si el parámetro es menor o igual.

bool operator ==(vInt);
Parámetros:
vInt: variable con la cual se va a comparar
Proceso realizado:

bool operator !=(vInt);
