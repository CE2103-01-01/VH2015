Clase vString

Ámbito privado:

vList<char>* word: puntero a una vList de char que almacena todos los caracteres que forman la cadena de texto. 



Ámbito público:

vString(std::string);
Parámetros:
std::string: cadena de texto que se desea guardar
Proceso realizado:
Recorre el parámetro recibido llenando una lista con el contenido.

~vString();
Proceso realizado:
Elimina la lista que forma la cadena de texto.

int operator +=(std::string);
Parámetros:
std::string: cadena de texto que se desea agregar.
Proceso realizado:
Lee el parámetro elemento a elemento y los agrega a la lista.

int operator +=(vString);
Parámetros: 
vString: cadena de texto que se desea agregar.
Proceso realizado:
Lee la lista de caracteres del parámetro y  los agrega uno a uno a la lista.

int operator =(vString);
Parámetros:
vString: cadena de texto que se desea asignar.
Proceso realizado:
Si la lista de caracteres no es nula, borra su contenido, lee el del parámetro e inserta la cadena elemento a elemento. Si la lista de caracteres es nula, la crea y le asigna los elementos del parámetro.

int operator =(std::string);
Parámetros:
std::string: cadena de texto que se desea asignar.
Proceso realizado:
Si la lista de caracteres no es nula, borra su contenido, lee el parámetro e inserta elemento a elemento. Si la lista de caracteres es nula, la crea y le asigna los elementos del parámetro.

char operator [](int);
Parámetros:
int: posición en la cadena.
Proceso realizado:
Se recorre la cadena con un iterador hasta encontrar la posición que indica el parámetro. Se retorna dicha posición.

int operator --();
Proceso realizado:
Elimina elemento a elemento todo el contenido de la lista de caracteres, vaciando el vString.

vList<char> operator !();
Proceso realizado:
Retorna la lista de caracteres que contiene la cadena.

bool operator==(std::string);
Parámetros:
std::string: cadena con la cual se va a comparar.
Proceso realizado:
Analiza la longitud y los elementos del parámetro, retorna false si hay diferencias.

bool operator==(vString);
Parámetros:
vString: objeto con el cual se va a comparar.
Proceso realizado:Analiza la longitud y los elementos del parámetro, retorna false si hay diferencias.
